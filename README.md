# Titanic Survival Prediction

Predicting survival of Titanic passengers using Machine Learning and Deep Learning.

## Dataset
- Source: [Kaggle Titanic Competition](https://www.kaggle.com/c/titanic)
- 891 training samples, 418 test samples

## Features Used
- Title extracted from Name
- Family Survival rate
- Family Size
- Fare and Age bins
- Passenger Class

## Models
| Model | Accuracy |
|---|---|
| ANN (PyTorch) | 76% |
